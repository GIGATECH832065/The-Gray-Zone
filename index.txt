            <!-- Step 1 -->
            <div id="check-privacy" class="section bg-black bg-opacity-70 p-6 rounded-lg mb-8">
                <div class="flex items-start mb-4">
                    <div class="step-number">1</div>
                    <h3 class="text-2xl">Check Privacy Settings</h3>
                </div>
                <div class="pl-14">
                    <p class="mb-4">If Location Services are grayed out in Windows 11 Home Edition, first check your privacy settings:</p>
                    <ul class="list-disc pl-6 space-y-2 mb-6">
                        <li>Open Settings (Windows + I) > Privacy & security > Location.</li>
                        <li>Ensure Location services and Let apps access your location are toggled on.</li>
                        <li>If grayed out, proceed to the next steps.</li>
                    </ul>
                    <div class="note mb-6">
                        <p class="font-bold">NOTE:</p>
                        <p>If you see "Some settings are managed by your organization," this indicates a policy or registry restriction, even on a personal PC.</p>
                    </div>
                    
                </div>
            </div>
            
            <!-- Step 2 -->
            <div id="geolocation" class="section bg-black bg-opacity-70 p-6 rounded-lg mb-8">
                <div class="flex items-start mb-4">
                    <div class="step-number">2</div>
                    <h3 class="text-2xl">Verify Geolocation Service</h3>
                </div>
                <div class="pl-14">
                    <p class="mb-4">Check if the geolocation service is running:</p>
                    <ul class="list-disc pl-6 space-y-2 mb-6">
                        <li>Press Windows + R, type services.msc, and press Enter.</li>
                        <li>Find Geolocation Service, right-click, and select Properties.</li>
                        <li>Set Startup type to Automatic and ensure Service status is Running.</li>
                        <li>Click Start if it's stopped.</li>
                        <li>Restart your PC and check if Location Services is enabled.</li>
                    </ul>
                  
                </div>
            </div>
            
            <!-- Step 3 -->
            <div id="registry" class="section bg-black bg-opacity-70 p-6 rounded-lg mb-8">
                <div class="flex items-start mb-4">
                    <div class="step-number">3</div>
                    <h3 class="text-2xl">Edit Registry Settings</h3>
                </div>
                <div class="pl-14">
                    <p class="mb-4">Since Windows 11 Home lacks Group Policy Editor, use the Registry Editor to fix restrictions:</p>
                    <div class="code-block mb-4 pl-6">
                        <p>Press Windows + R, type regedit, and press Enter.</p>
                    </div>
                    <ul class="list-disc pl-6 space-y-2 mb-6">
                        <li>Navigate to: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\lfsvc\TriggerInfo</li>
                        <li>Right-click the key named 3 and select Delete.</li>
                        <li>Navigate to: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\LocationAndSensors</li>
                        <li>If DisableLocation exists, double-click it and set its value to 0.</li>
                        <li>If it doesn't exist, create a new DWORD (32-bit) Value named DisableLocation and set it to 0.</li>
                        <li>Restart your PC and check Settings.</li>
                    </ul>
                    <div class="note mb-6">
                        <p class="font-bold">WARNING:</p>
                        <p>Always back up your registry before making changes. Incorrect registry edits can cause system instability.</p>
                    </div>
                    <div class="flex justify-center">
                        <button onclick="window.open('reg.html', '_blank')" class="backup-btn px-6 py-2 rounded transition duration-300 ease-in-out">
                            <i class="fas fa-database mr-2"></i> HOW TO BACKUP
                        </button>
                    </div>
                    
                </div>
            </div>
            
            <!-- Step 4 -->
            <div id="management" class="section bg-black bg-opacity-70 p-6 rounded-lg mb-8">
                <div class="flex items-start mb-4">
                    <div class="step-number">4</div>
                    <h3 class="text-2xl">Check for Management Software</h3>
                </div>
                <div class="pl-14">
                    <p class="mb-4">Some security or parental control software can restrict Location Services:</p>
                    <ul class="list-disc pl-6 space-y-2 mb-6">
                        <li>Review installed apps (e.g., antivirus, VPNs, or enterprise tools) and temporarily disable or uninstall them to test.</li>
                        <li>If using Windows Defender, it's unlikely to cause this issue.</li>
                    </ul>
                    
                </div>
            </div>
            
            <!-- Step 5 -->
            <div class="section bg-black bg-opacity-70 p-6 rounded-lg mb-8">
                <div class="flex items-start mb-4">
                    <div class="step-number">5</div>
                    <h3 class="text-2xl">Run System File Checker (SFC) and DISM</h3>
                </div>
                <div class="pl-14">
                    <p class="mb-4">Corrupted system files may cause the issue. Run these commands in Command Prompt as Administrator:</p>
                    <div class="code-block mb-4">
                        <p>sfc /scannow</p>
                    </div>
                    <div class="code-block mb-6">
                        <p>DISM /Online /Cleanup-Image /CheckHealth</p>
                        <p>DISM /Online /Cleanup-Image /ScanHealth</p>
                        <p>DISM /Online /Cleanup-Image /RestoreHealth</p>
                    </div>
                    <p class="mb-6">Restart your PC and check Location Services.</p>
                    
                </div>
            </div>
            
            <!-- Step 6 -->
            <div class="section bg-black bg-opacity-70 p-6 rounded-lg mb-8">
                <div class="flex items-start mb-4">
                    <div class="step-number">6</div>
                    <h3 class="text-2xl">Check for Windows Updates</h3>
                </div>
                <div class="pl-14">
                    <ul class="list-disc pl-6 space-y-2 mb-6">
                        <li>Go to Settings > Windows Update > Check for updates.</li>
                        <li>Install any pending updates, as they may fix bugs affecting Location Services.</li>
                        <li>Restart your PC after updating.</li>
                    </ul>
                    
                </div>
            </div>
            
            <!-- Step 7 -->
            <div class="section bg-black bg-opacity-70 p-6 rounded-lg mb-8">
                <div class="flex items-start mb-4">
                    <div class="step-number">7</div>
                    <h3 class="text-2xl">Use a Registry Fix File</h3>
                </div>
                <div class="pl-14">
                    <p class="mb-4">If manual registry edits are daunting, download a trusted .reg file to enable Location Services:</p>
                    <ul class="list-disc pl-6 space-y-2 mb-6">
                        <li>Visit a reputable source like elevenforum.com for a .reg file to enable Location Services (e.g., set DisableLocation to 0).</li>
                        <li>Double-click the .reg file, approve the merge, and restart your PC.</li>
                    </ul>
                    <div class="note mb-6">
                        <p class="font-bold">CAUTION:</p>
                        <p>Only download registry files from trusted sources. Malicious registry files can compromise your system.</p>
                    </div>
                </div>
            </div>
            
            <!-- Step 8 -->
            <div class="section bg-black bg-opacity-70 p-6 rounded-lg mb-8">
                <div class="flex items-start mb-4">
                    <div class="step-number">8</div>
                    <h3 class="text-2xl">System Restore or Repair Install</h3>
                </div>
                <div class="pl-14">
                    <p class="mb-4">If the issue started after a recent update:</p>
                    <ul class="list-disc pl-6 space-y-2 mb-6">
                        <li>Open Control Panel > Recovery > Open System Restore and select a restore point before the issue began.</li>
                        <li>Alternatively, perform a repair install using a Windows 11 ISO:
                            <ul class="list-disc pl-6 mt-2 space-y-2">
                                <li>Download the ISO from Microsoft's website.</li>
                                <li>Double-click the ISO, run setup.exe, and choose to repair without deleting files.</li>
                            </ul>
                        </li>
                    </ul>
                    
                </div>
            </div>
            
            <!-- Step 9 -->

                    <div class="step-number">9</div>
                    <h3 class="text-2xl">Create a New User Account</h3>
                </div>
                <div class="pl-14">
                    <p class="mb-4">If the issue persists, your user profile may be corrupted:</p>
                    <ul class="list-disc pl-6 space-y-2 mb-6">
                        <li>Create a new local admin account: Settings > Accounts > Family & other users > Add account.</li>
                        <li>Sign in to the new account and check if Location Services works.</li>
                        <li>If it does, transfer your files and delete the old account.</li>
                    </ul>
                    <div class="flex justify-center">

                </div>

                    <li>If you see persistent "managed by your organization" messages, check for work/school accounts in Settings > Accounts > Access work or school and disconnect them.</li>
                    <li>If none of these work, consider resetting Windows 11 (Settings > System > Recovery > Reset PC), but back up your data first.</li>
